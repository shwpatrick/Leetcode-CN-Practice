# Sol.1   

語言: C++  
解題思路：暴力解遍歷後排序  
時間複雜度：O(n + logn)    
空間複雜度：O(1)

```cpp
class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        for (int i = 0; i < nums.size(); i++) {
            nums[i] = nums[i] * nums[i];
        }
        sort(nums.begin(), nums.end());
        return nums;
    }
};
```
